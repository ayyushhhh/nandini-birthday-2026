# Nandini Birthday Website

This is a complete static website. You can deploy the whole `nandini-birthday`
folder on Netlify, Vercel, GitHub Pages, or any normal static hosting service.

## Open Locally

Open `index.html` in a browser, or run a small local server from this folder:

```powershell
python -m http.server 4173
```

Then visit:

```text
http://localhost:4173
```

The birthday song starts after the first button tap because browsers block
automatic music until the visitor interacts with the page.
